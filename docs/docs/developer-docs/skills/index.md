# Skills

IPA skills are Claude Code slash commands that automate infrastructure provisioning, deployment, and project management. Each skill is invoked with `/ipa-{name}` and performs a specific operation in the IPA workflow.

Skills are divided into three categories:

- **[Lifecycle Skills](./lifecycle-skills/)** — Process skills that orchestrate the IPA workflow from initialization through deployment and teardown
- **[Stack Skills](./stack-skills/)** — Per-stack reference for the CloudFormation stacks that `/ipa-compose` assembles into deployment patterns
- **[Author Skills](./author-skills/)** — Tools for creating and updating stack skills and pattern definitions
