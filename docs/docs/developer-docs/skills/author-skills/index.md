# Author Skills

Author skills create and update the stack skills and pattern definitions that `/ipa.compose` consumes. Use these skills to extend IPA with custom infrastructure.

| Skill | Purpose |
|-------|---------|
| [/ipa.author.stack](./ipa-author-stack.md) | Create or update stack skills, CloudFormation templates, and pattern definitions |
| [/ipa.author.guide](./ipa-author-guide.md) | Write documentation guides conforming to the Guide Format Standard |
