# util/

Shared utilities. See README.md for usage.

| Utility | Purpose |
|---------|---------|
| `PathUtil` | Resolve paths relative to lib/repo root |
| `JinjaUtil` | Render Jinja2 templates |
| `PynamodbUtil` | Environment-aware table naming |
| `AgentCoreUtil` | Construct runtime URLs from ARNs |
| `ObservabilityUtil` | Structured error logging, CloudWatch metric emission, Bedrock token parsing |
