# Titanic Dataset

Passenger manifest from the RMS Titanic, used as a demo dataset for the Innovation Patterns walkthrough.

## Source

This is a widely available public dataset derived from the Encyclopedia Titanica. The copy used here was sourced from common ML tutorial repositories.

## Format

`walkthrough_titanic.csv` — 1,309 rows with header, 14 columns:

| Column | Description |
|--------|-------------|
| pclass | Passenger class (1, 2, 3) |
| survived | Survival (0 = no, 1 = yes) |
| name | Passenger name |
| sex | Gender |
| age | Age in years |
| sibsp | Number of siblings/spouses aboard |
| parch | Number of parents/children aboard |
| ticket | Ticket number |
| fare | Passenger fare |
| cabin | Cabin number |
| embarked | Port of embarkation (S, C, Q) |
| boat | Lifeboat number |
| body | Body identification number |
| home.dest | Home/destination |
